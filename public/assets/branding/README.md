# Recursos Gráficos del Proyecto (Branding)

Este directorio está reservado para los recursos gráficos oficiales de la plataforma gubernamental "inspira-mas-feliz".

Aquí deben residir los siguientes archivos vectoriales (.svg):

1. **`logo-gobierno.svg`**: Logo oficial de la entidad gubernamental.
2. **`logo-inspira.svg`**: Logotipo de la iniciativa "Inspira".
3. **`logo-mas-feliz.svg`**: Logo principal de la plataforma "Más Feliz".

Asegúrese de subir versiones optimizadas y con el formato SVG estándar para garantizar su correcto renderizado en la interfaz.
